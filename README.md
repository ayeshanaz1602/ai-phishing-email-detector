# 🛡️ AI Phishing Email Detection System

A Machine Learning-based cybersecurity tool that detects phishing emails using NLP, suspicious URL analysis, and threat intelligence techniques.

## 🚀 Key Features

- Machine Learning-based phishing detection
- Suspicious URL analysis
- Threat-level classification
- Explainable phishing reasons
- Scam keyword detection
- Confusion matrix evaluation
- 97%+ model accuracy
